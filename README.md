# Bin Askar Technology | بن عسكر للتقنية

Corporate website for **Bin Askar Technology** — bilingual (Arabic/English) Next.js site for technology consulting led by Abdullah Bin Askar.

- **Production:** [binaskar.org](https://binaskar.org)
- **Stack:** Next.js 16, TypeScript, Tailwind CSS
- **Version:** 1.0.0

## Development

```bash
npm install
npm run dev        # http://localhost:3000/ar
npm run build
npm run start
npm run test
npm run test:e2e
```

Copy `.env.example` to `.env.local` and configure PrivateEmail SMTP (`SMTP_USER`, `SMTP_PASS`) plus `CONTACT_EMAIL` and `NEXT_PUBLIC_SITE_URL`.
